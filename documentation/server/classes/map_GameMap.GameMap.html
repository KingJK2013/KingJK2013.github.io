Static getRandomTileWith

  • getRandomTileWith(options): Tile

  • This method iterates through tiles randomly until it finds a tile that meets the specified criteria. If the method cannot find a suitable Tile object after 7500 iterations, it returns undefined.

    Parameters

    • options: {
          avoidResourceTiles?: boolean;
          avoidTileTypes?: string[];
          onAdditionalTileTypes?: boolean;
          tempRange?: [number, number];
          tileTypes?: string[];
      }
      • Optional avoidResourceTiles?: boolean

        (Optional) A boolean value that indicates if we allow the random tile to be an existing resource tile(E.g. coal, horses, fish, ect.). Default value = FALSE

      • Optional avoidTileTypes?: string[]
      • Optional onAdditionalTileTypes?: boolean

        (Optional) A boolean value that indicates if we allow the random tile to contain more than 1 tile type. (E.g. a forest). Default value = FALSE

      • Optional tempRange?: [number, number]

        (Optional) A tuple of two numbers that specify the temperature range for the generated Tile object. If not provided, the default temperature range is [0, 100].

      • Optional tileTypes?: string[]

        The tiletypes of tiles that can be randomly picked.

    Returns Tile

    A Tile object that meets the specified criteria, or undefined if no such Tile is found.
